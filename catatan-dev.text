PANDUAN INSTALASI AWAL (Hanya Sekali)

Install Git git version 2.53.0.windows.1: Download di git-scm.com, install (Next > Next sampai selesai).

Install Python Python 3.12.7: Download di python.org. PENTING: Saat install, centang kotak "Add Python to PATH" di bagian bawah sebelum klik Install.

Install Node.js (Untuk Web v24.13.0): Download di nodejs.org (pilih versi LTS).

Ambil Project:

Buat folder baru (misal: Kerjaan-NAR).

Klik kanan di dalam folder itu > Open Git Bash Here (atau Terminal).

Ketik: git clone https://github.com/loehak5/instatools.git

CARA KERJA SETIAP HARI

Buka folder instatools.

Cari file bernama Start-Instatool.bat.

Klik 2x file tersebut.

Tunggu sebentar (dia akan otomatis update), lalu aplikasi akan terbuka.
