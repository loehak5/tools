PANDUAN INSTALASI 

Install Git git version 2.53.0.windows.1: Download di git-scm.com. 
-disediakan tombol downloadnya
-install (Next > Next sampai selesai). 

Install Python Python 3.12.7: Download di python.org. 
-disediakan tombol downloadnya
-PENTING: Saat install, centang kotak "Add Python to PATH" di bagian bawah sebelum klik Install.

Install Node.js (Untuk Web v24.13.0): 
-disediakan tombol downloadnya


Ambil Project:

Buat folder baru (misal: Kerjaan).

Klik kanan di dalam folder itu > Open Git Bash Here (atau Terminal).

Ketik: git clone https://github.com/loehak5/instatools.git

CARA KERJA SETIAP HARI

Buka folder instatools.

Running maintenance windows (hanya untuk sekali perintah).
-Cari file bernama Aktifkan-Maintenance.bat.
-Klik 2x file tersebut.
-Tunggu sebentar (dia akan otomatis update), lalu aplikasi akan terbuka. Jika sudah selesai maka akan menutup sendiri dan tidak perlu menjalankan lagi disetrap menyalakan komputerlagi.

Running aplikasi di windows
-Cari file bernama Mulai-tools.bat.
-Klik 2x file tersebut.
-Tunggu sebentar (dia akan otomatis update), lalu aplikasi akan terbuka.
